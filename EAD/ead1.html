  • O que é variável? Qual a sua finalidade?
  • Quais os tipos de variáveis podemos ter?

Em JavaScript, uma variável é um identificador usado para armazenar valores, como números, strings, objetos, funções e outros tipos de dados. A finalidade das variáveis é tornar o código mais legível e mantível, fornecendo nomes significativos aos valores que são usados ou manipulados no programa.

O objetivo das variáveis ​​é armazenar dados temporariamente para que você possa consultá-los posteriormente em seu programa. Isso permite organização, legibilidade e manutenção mais fáceis do código.

Você pode declarar uma variável em JavaScript usando a palavra-chave let, const ou var. Aqui estão alguns exemplos:

1 let meuNome = 'Joao' ;

2 let idade = 25 ;

3 let euExisto = true ;

4

5 const PI = 3.14 ;

6 var salario ;

7

8 salario = 1500 ;

9 salario = 2500 ;

Let: Em algumas linguagens de programação, você pode criar variáveis que só podem ser usadas dentro de um pequeno pedaço de código, como um bloco, uma linha ou uma expressão.

Const: Define uma função que retorna sempre o mesmo resultado, ou seja, um valor constante.

Var: é uma palavra-chave em JavaScript usada para declarar variáveis, mas foi em grande parte substituída.

Ao trabalhar com variáveis, é essencial seguir as melhores práticas, como usar nomes significativos e declarar variáveis ​​perto de onde elas são usadas. Isso ajudará a tornar seu código mais legível e fácil de manter.

Para que serve os comandos alert, prompt e document.write? Quais as suas diferenças?

alert

A função é usada para exibir uma mensagem simples ao usuário em uma caixa de diálogo. A caixa de diálogo contém um botão OK e o usuário só pode descartá-la clicando em OK. Nenhuma entrada é possível e a página não é atualizada ou alterada.

1 alert ( ' Hello world! ' ) ;

prompt()

A função é usada para solicitar entrada do usuário em uma caixa de diálogo. A caixa de diálogo contém um campo de entrada de texto e botões OK e Cancelar. Se o usuário clicar em OK, o valor de entrada será retornado; se o usuário clicar em Cancelar, será cancelado.

1 let userName = prompt ( ' Digite seu nome: ' ) ;

2 alert ( ' Hello, ' , userName , ' ! ' ) ;

document.write()

A função é usada para escrever conteúdo HTML diretamente no documento atual. Pode alterar o conteúdo e a estrutura da página. Geralmente não é recomendado para código de produção, pois pode causar problemas na renderização da página e na experiência do usuário.

1 document.write ( ' Hello World ! ' ) ;
